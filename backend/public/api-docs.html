🏪 API da Barbearia do Além

✅ API Backend Funcionando!

Versão: 2.0.0 | Porta: 3000 | Status: 🟢 Online

Bem-vindo à API da Barbearia do Além. Esta API fornece endpoints para gerenciar agendamentos, clientes e serviços.

Endpoints Disponíveis:

🔐 Autenticação

POST /api/admin/login - Login do administrador

POST /api/admin/logout - Logout do administrador

GET /api/admin/verificar-sessao - Verificar sessão ativa

⚙️ Serviços

GET /api/servicos - Listar todos os serviços disponíveis

👥 Clientes

POST /api/clientes - Criar um novo cliente

GET /api/clientes - Listar todos os clientes

📅 Agendamentos

GET /api/agendamentos - Listar todos os agendamentos

POST /api/agendamentos - Criar um novo agendamento

PATCH /api/agendamentos/:id - Atualizar status de um agendamento

DELETE /api/agendamentos/:id - Cancelar agendamento

🚫 Datas Bloqueadas

GET /api/datas-bloqueadas - Listar datas bloqueadas

POST /api/datas-bloqueadas - Bloquear uma data

DELETE /api/datas-bloqueadas/:data - Desbloquear uma data

📊 Status e Logs

GET /api/status - Status da aplicação

GET /api/admin/logs - Logs do sistema

🌐 Links Úteis

Frontend da Aplicação: http://localhost:4200

Painel Admin: http://localhost:4200/admin

Login Admin: http://localhost:4200/login
